About us

codiez is a software development company founded in 2008, originally under the name ZyeLabs, since then we have rebranded. We have a wealth of experience with various technologies and software development languages. We believe there is no one size fits all solution. A true craftsman should know the right tool for the job, and we pride ourselves in selecting the best tools to complete your project.

At codiez we believe software should be written with agility, using proven techniques that ensure the best quality of code such as TDD (Test Drive Development) and BDD (Behavior Driven Development).
